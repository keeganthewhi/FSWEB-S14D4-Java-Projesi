## About

Covers abstract classes and interfaces in Java, two key mechanisms for defining contracts and achieving abstraction. This assignment demonstrates when to use each and how they enable flexible, modular design.

## Key Concepts

- Abstract classes and abstract methods
- Interfaces and `implements` keyword
- Multiple interface implementation
- Abstract classes vs. interfaces: when to use which
